Jdao: Java Data Access Object

Description: Web application for designing relational databases and rapidly generating editable form/grid web pages from data. With source codes, this web application can serve as a foundation of any other data-driven web applications. It needs to be installed on a Java web application container such as Tomcat .
Download jdao.war
