Facial Attendance

Description
This Attendance add uses member's face image as the Id to administrate attendance of meetings, classes or other group activities. Targeted users are teachers, team leaders and organizers of other informal group activities. Analyzing the image taken by user on the first-time meeting, this app is able to detect the faces of members and use them as the Id to build a list of group members. On each following meeting, user can tap the face icon to take attendance without remembering people's names or calling people's name explicitly.
