Privacy Policy about Using "Facial Attendance" App

  1. This app is used to take attendance of classes, meetings or any group activities. Absolutely no data will go outside of the device unless user sends data out explicitly.
  2. This app does not have any back-end and will not collect any user information or sensitive data.
  3. The app will not read any data out of the memory except for those generated by the app itself.
  4. The app will not receive any data from outside.
  5. The app optionally request permission to use camera to catch instant images of students under user's guidance to reduce the number of manual operations for building a student list. But user can reject the request without lowering the app's essential functionality.

关于使用扫描评分的隐私政策

  1. 该应用被用来做教学、会议或任何群体活动考勤。 绝对没有数据 被自动发送到设备的外部。
  2. 此应用程序没有任何后端，也不会收集任何用户信息或敏感数据。
  3. 该应用程序将不会从内存中读取任何非本程序产生的数据数据。
  4. 该应用程序将不会从外部接收任何数据。
  5. 该应用会选择请求使用摄像头在用户操作下拍摄照片以减少用于创建一个学生花名册的手动操作。 但是用户拒绝摄像头许可而不会明显降低该应用的功能
