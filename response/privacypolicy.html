Privacy Policy about Using "Response" App

  1. This app facilitates the data exchange between various devices in a wireless local area network. User can control communication partner's permissions of sending and receiving. Absolutely no data will go beyond the boundary of the wireless local area network.
  2. This app does not have any back-end and will not collect any user information or sensitive data.
  3. The app will not read any unnecessary information out of the memory.
  4. The app will not send any file to communication partners without user's explicit manual operation.
  5. The app optionally request permission to use camera to catch instant images under user's guidance to reduce the number of manual operations for sending instant images to communication partners. But user can reject the request without lowering the app's essential functionality.

关于使用【应答】的隐私政策

  1. 此应用程序旨在开通无线局域网中各种设备之间的数据交换。 用户可以控制通讯各方发送和接收的权限。 绝对没有数据会送出所处的无线局域网之外。
  2. 此应用程序没有任何后端，也不会收集任何用户信息或敏感数据。
  3. 该应用程序不会从内存中读取任何不必要的信息。
  4. 在任何情况下，不经由用户明确的手动操作，该应用程序将不会自动向通讯伙伴发送任何文件。
  5. 该应用会选择请求使用摄像头在用户操作下拍摄照片以减少用于向通讯伙伴发送即时图像的手动操作。 但是用户拒绝摄像头许可而不会明显降低该应用的功能
