Enquire-Response

Description
Class/meeting/group coordinator and file synchronizer: administrating quizzes, surveys, data collections and keeping files synchronized cross platforms: iphone, ipad, android devices, Macbook and Windows PCs with minimum bit transmissions.
Download Desktop Java version for Windows & Mac
Android version as official Google Play Store listing: https://play.google.com/store/apps/details?id=com.systemsonweb.response (tap and scan the QRcode ➚)
Product logo
How to use
  1. Download the zip archive, unzip it
  2. Copy the generated folder to all PCs and install mobile version on mobile devices.
  3. There are two modes: Enquirer and Responder. Enquirer waits and receives submissions of quiz answers and file uploads, as well as offers file distributions. Responder submits files, answers and download files or folders from the Enquirer
