Description Class/meeting/group coordinator and file synchronizer: administrating quizzes, surveys, data collections and keeping files synchronized cross platforms: iphone, ipad, android devices, Macbook and Windows PCs.
Download Executable desktop.zip
Other versions are coming soon
How to use
  1. Download the zip archive, unzip it
  2. Copy the folder to different PCs.
  3. There are two modes: Enquirer and Responser. Enquirer waits and receives for submissions of quiz answers and file uploads, as well as offers file uploads. Responser submits files, answers and download files or folders from the Enquirer
