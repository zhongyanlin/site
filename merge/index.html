Merging Blackboard Gradebook with Other Score Reports

Instructions:
  1. Download the grade book from Blackboard Grade Center with Work Offline option using comma as the separator, copy-paste the file content to the left textbox and parse the text (see 1st Parse button)
  2. Copy-paste/download a score report from the other software such as google form/github/zybooks, copy-paste the file content to the right textbox and the parse the text(see 2nd Parse button)
  3. Click the emerging Match button, and resolve possible mismatches and duplicates in the emerging dialog window
  4. On the heading of the second resulted table, decide the matching columns or set unwanted column's heading to blank (the emerging Null Heading button can help). Click the emerging Merge button next to the Match button.
  5. If the data are merged correctly, click the emerging CSV button under the merged table. This will generate a csv file format of merged data
  6. Copy-paste generated text inside the bottom textbox to override original downloaded file and upload the modified file back to Blackboard
  7. Optionally, for the convenience of transferring grades to the banner system, downloading file again(needs BB to do calculation) and parse again, you may use Swap Column to bring the Weighted Total or Final Grade column closer to student name column
