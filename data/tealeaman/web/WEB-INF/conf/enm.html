When you design a data entry form and intend to print it out on paper and let your clients fill out it by pen, you may use consecutive underscore character strings, such as _______, intuitively to mark blank field for your clients to fill out. This program is just to convert such forms to web form. The source file must be HTML document. For this end, you need to perform the following pre-process before conversion: If your source file is a Microsoft Word doc file, please save it as a HTML web page file in Word, and if it is a PDF file, please go to http://access.adobe.com/access to convert it to a HTML web page file. If you don't have a HTML-form now, go ahead to click the "New" button anyway, a wizard will have a "MakeOne" button to enable you to create a HTML-format form instantly.

The following are the rules of converting HTML-format paper forms to web form:
  1. A selective item set with camma sign or blank character as separator, formatted as (1=poor, 2=good) or (1=poor 2=good), where the parentheses are required, in the file will be converted to a group of radios like this poor good, and this group of radios will keep replacing the following consecutive underscore patterns until the number of underscores is changed.
  2. A selective item set with new line or slash as separator formatted as
    (1=poor
    2=good)

    or (1=poor/ 2=good) will be converted to a select box like this , and this select box will keep replacing the following consecutive underscore patterns until the number of underscores is changed.
  3. If not replaced by radios or selects, a pattern of two consecutive underscores, i.e, ___, will be converted to a checkbox like this
  4. If not replaced by radios or selects, a pattern of three or more consecutive underscores enclosed in a square blacket, e.g.[________________________], will be converted to a textarea like this
  5. If not replaced by radios or selects, a pattern of three or more consecutive underscores enclosed by @, e.g.@___________________@, will be converted to a widget for file attachment like ⊕ file1.jpg ⊖ , clicking these ⊕, file name and ⊖ will start attaching more, opening file and deleting file functions, respectively.
  6. If not replaced by radios or selects, a pattern of three or more consecutive underscores, e.g._______, will be converted to a textbox like this
  7. Tables containing two consecutive blank cells but no consecutive underscore character pattern will be made editable and automatic row growing
