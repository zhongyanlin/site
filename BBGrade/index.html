Blackboard Batch Grade
This desktop Java Application grades the downloaded student's submissions of assignments and tests in a batch mode. It does automatic scanning and estimating of scores, load all submissions and estimated scores in a single window to allow user to correct scores generated by auto-grading. It also merges the grade sheets downloaded from classroom.github and classroom.google. All records are saved in a local database for further querying.
Download Executable: Desktop Java version for Windows/Mac/Linux

How to use:
  1. Download the zip archive, unzip it
  2. In the dist folder, double click the executable BBGrade.jar.
  3. Go to the Bb's Full Grade Center, select a specific submitted item of assignment, expands the top menu, selects "Assignment File Download" to download all submissions as a single zip archive, then go to "Work Offline" to download the grade sheet of this item as a csv file.
  4. Click the New menu item in BBgrade to allow it to find the downloaded archive and grade sheet from the username/Downloads folder, then enter reference/standard answers, click Parse and Scan to scan the student's submitted answers against reference/standard answers. In the pop-out window that loads all submissions with estimated scores, correct the scores and save all.
  5. Upload the modified Grade Sheet back to Blackboard
