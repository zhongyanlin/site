Blackboard Batch Grade
This desktop Java Application grades the downloaded student's submissions of assignments and tests in a batch mode. User goes to the Bb's Full Grade Center, selects a specific submitted item of assignment, expands the top menu, selects "Assignment File Download" to download all submissions as a single zip archive, then goes to "Work Offline" to download the grade sheet of this item as a csv file. User then starts the BBgrade to allow it to find the downloaded archive and grade sheet from the username/Downloads folder, then enters reference/standard answers, click Parse and Scan to scan the student's submitted answers against reference/standard answers entered by user (instructor or grader). All submissions with estimated scores will be loaded in a single Windows for user to check.
Download Executable: Desktop Java version for Windows/Mac/Linux

How to use:
  1. Download the zip archive, unzip it
  2. In the dist folder, double click the executable BBGrade.jar.
