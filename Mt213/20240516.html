Slide 2: Logic AND Title: Logic AND (Conjunction) Definition: Combining propositions using "AND" Symbol: ∧ (ampersand) or ">" Truth Table P: True / False Q: True / False P ∧ Q: (T, T), (T, F), (F, T), (F, F) (Explain each case) Example: "I will study AND I will pass the exam" Logical Equivalence: Commutative (P ∧ Q ≡ Q ∧ P) and Associative (P ∧ (Q ∧ R) ≡ (P ∧ Q) ∧ R)
Slide 3: Logic OR Title: Logic OR (Disjunction) Definition: Combining propositions using "OR" Symbol: ∨ (vee) or "∨" Truth Table P: True / False Q: True / False P ∨ Q: (T, T), (T, F), (F, T), (F, F) (Explain each case) Example: "I will study OR watch a movie" Logical Equivalence: Commutative (P ∨ Q ≡ Q ∨ P) and Associative (P ∨ (Q ∨ R) ≡ (P ∨ Q) ∨ R)
Slide 4: Logic NOT Title: Logic NOT (Negation) Definition: Inverting the truth value of a proposition Symbol: ¬ (not) or '¬' Truth Table P: True / False ¬P: False / True Example: "I am not tired" Logical Equivalence: Double negation (¬(¬P) ≡ P)
Slide 5: Implication Title: Implication (Conditional) Definition: If P, then Q (P → Q) Symbol: → or "implies" Truth Table P: True / False Q: True / False P → Q: (T, T), (T, F), (F, T), (F, F) (Explain each case) Example: "If it rains, the streets will be wet" Logical Equivalence: Material Implication (P → Q ≡ ¬P ∨ Q)
Slide 6: Deduction Title: Deduction Definition: Inference of a conclusion from one or more premises Rules: Modus Ponens, Modus Tollens, etc. Modus Ponens: If P → Q and P, then Q Modus Tollens: If P → Q and ¬Q, then ¬P Example: "If I study, I will pass. I studied. Therefore, I will pass."
Slide 7: Contrapositive Title: Contrapositive Definition: If P → Q, then ¬Q → ¬P Importance: Contrapositives are logically equivalent to the original implication Example: "If it's sunny, the pool is open. Therefore, if the pool is not open, it's not sunny."
Slide 8: Promise Title: Promise in Propositional Logic Definition: A statement that predicts a future event Example: "If I finish my homework, I will play video games" Importance: Understanding implications and conditions in real-life scenarios
Slide 9: Truth Table (Summary) Title: Truth Table Purpose: To determine the truth value of a proposition based on its components Components: AND, OR, NOT, implication Demonstrate the truth table for a compound proposition (e.g., (P ∧ ¬Q) → (R ∨ P))
